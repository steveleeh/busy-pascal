# 🏛️ AI 历史演进实验室 (AI Evolutionary Lab)

欢迎来到你的 AI 历史演进实验室！这里不是一个简单的代码库，而是一条重温人类 AI 探索历程的“时间走廊”。

在这里，我们将通过**五个核心的实战项目**，逐一重现并解决过去 20 年间限制 AI 发展的致命瓶颈，最终通往现代大模型与通用智能体（Agent）的摩天大楼。

## 🛠️ 如何开始？

每个项目都包含：
1. `scratch_*.py`：**纯 Python 从零手写实现**（无任何第三方库依赖，帮你吃透数学与代码底层逻辑）。
2. `sklearn_*.py` 或框架版：工业级第三方库（如 scikit-learn, PyTorch）的对比实现，带你体验工程抽象。
3. `README.md`：极具深度的**局限性分析、数学推导与下一步演进思考**。
